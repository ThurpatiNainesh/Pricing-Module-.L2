# Ride Pricer Node

A configurable pricing module backend in Node.js and MongoDB for calculating ride fares.

## Features

- Configurable pricing rules based on:
  - Day of week
  - Distance slabs
  - Time multiplier slabs
  - Waiting charges
- Audit log for config changes
- Pricing calculation API

## Run Locally

```bash
npm install
npm start
